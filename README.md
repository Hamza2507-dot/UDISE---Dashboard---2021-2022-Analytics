# UDISE---Dashboard---2021-2022-Analytics
Dashboard analyzing UDISE+ 2021–22 enrolment data, highlighting gender parity and state-wise trends.

Project Overview;
    This project analyzes enrolment data from the UDISE+ 2021–22 (Unified District Information System for Education, Government of India). The dashboard was created in Excel to highlight enrolment trends across states and union territories, focusing on gender parity, enrolment distribution, and state-wise comparisons.

Data Source;
    UDISE+ 2021–22 official portal (Government of India) Raw data was downloaded and cleaned before visualization.

Process;
   
    Data Cleaning
    Removed inconsistencies and arranged raw tables. Sorted enrolment figures by state and education level.

    Data Preparation
    Organized enrolment by categories: Pre-Primary, Primary, Upper Primary, Secondary, and Higher Secondary. Calculated Gender Gap and Gender Parity Index (GPI).

    Dashboard Creation (Excel)
    Used pivot tables, charts, and conditional formatting. Designed a clear, interactive view of enrolment trends.

Key Insights;
    
    India’s total enrolment (Pre-Primary to Higher Secondary): 88.27 million.
    Gender Parity Index (GPI) at the national level: 0.80, showing fewer girls enrolled compared to boys.
    States like Meghalaya and Mizoram show near parity or higher female enrolment.
    Large states such as Uttar Pradesh, Maharashtra, and Madhya Pradesh contribute significantly to overall enrolment.

Tools Used;
   
    Microsoft Excel
        
        Pivot Tables
        Charts (Bar, Line, Pie)
        Conditional Formatting
        Data Cleaning & Sorting

How to Use;
    
    Download the Excel file from this repository.
    Open in Microsoft Excel (2016 or later recommended).
    Explore the dashboard tabs to view enrolment trends and gender parity analysis.
